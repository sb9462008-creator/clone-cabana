This is a Vite + React project using React Router for routing.

## Getting Started

First, install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open the URL printed in the terminal to see the result.

You can start editing the page by modifying `src/pages/Home.jsx`. The page auto-updates as you edit the file.

Global styles live in `src/app/globals.css` and are imported in `src/main.jsx`.

## Learn More

To learn more about Vite, take a look at the following resources:
